Dados abertos sobre o 25 de Abril


O Arquivo.pt fez uma recolha especial de páginas Web que têm referências ao 25 de Abril.

A lista das páginas identificadas, no total 12.650, está disponível no portal de dados abertos Dados.gov.


Dados abertos: lista de páginas Web acerca do 25 de Abril.


Para recolher uma quantidade considerável de páginas sobre um tema usamos uma metodologia humana e automática. Primeiro, criamos uma lista de termos de pesquisa, como aqueles que qualquer pessoa usaria para pesquisar na Internet. Em seguida, colocamos esses termos no serviço automático de pesquisa Bing Search API, de onde obtemos uma lista de resultados.


O primeiro objetivo, ao recolher um conjunto de páginas acerca do 25 de Abril, é sensibilizar os cidadãos para a importância de preservar os conteúdos publicados na Web, antes que estes desapareçam. O segundo, é melhorar o acervo arquivado sobre este tema e ser útil para a memória histórica e a investigação.
